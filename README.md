# Library Rental System

A console-based Library Rental System developed in Java using Object-Oriented Programming (OOP) principles and the Java Collections Framework. This application allows users to manage books by adding, searching, renting, returning, deleting, and viewing book records through a menu-driven interface.

## Features

- Add a new book
- Search books by title
- Search books by author
- Rent a book
- Return a book
- Delete a book
- Display all books
- Display available books
- Display rented books
- Display total number of books
- Prevent duplicate Book IDs
- Basic input validation using exception handling

##  Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (`ArrayList`)
- Exception Handling
- Scanner Class

##  How to Run

### Compile

```bash
javac LibraryRentalSystem.java
```

### Run

```bash

```
